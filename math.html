核心概念

拉格朗日乘數法

求解約束最佳化問題的標準方法：

$$\mathcal{L} = f(x_1, x_2, ...) - \lambda(g(x_1, x_2, ...) - c)$$

一階條件：$\frac{\partial \mathcal{L}}{\partial x_i} = 0$ 且 $\frac{\partial \mathcal{L}}{\partial \lambda} = 0$

邊際產出（MP）

增加一單位投入所增加的產出量：

$$MP_x = \frac{\partial Q}{\partial x}$$

最佳化條件：$\frac{MP_x}{P_x} = \frac{MP_y}{P_y}$

技術替代率（MRTS）

維持相同產出下，兩要素的替代比率：

$$MRTS_{xy} = \frac{MP_x}{MP_y} = \frac{\partial Q / \partial x}{\partial Q / \partial y}$$

等產量線的斜率絕對值

替代彈性（σ）

衡量要素比例變動對技術替代率的反應：

$$\sigma = \frac{d \ln(x/y)}{d \ln(MRTS_{xy})}$$

σ = 1：Cobb-Douglas；σ = ∞：完全替代

齊次函數

滿足以下性質的函數：

$$f(tx_1, tx_2, ...) = t^k f(x_1, x_2, ...)$$

k為齊次度，反映規模報酬特性

Cobb-Douglas函數

標準形式：

$$Q = A x_1^{\alpha_1} x_2^{\alpha_2} ... x_n^{\alpha_n}$$

齊次度 = $\alpha_1 + \alpha_2 + ... + \alpha_n$

關鍵公式總結

1. 最佳化條件

$$\frac{MP_{x_1}}{P_{x_1}} = \frac{MP_{x_2}}{P_{x_2}} = ... = \frac{MP_{x_n}}{P_{x_n}} = \lambda$$

2. MRTS計算

$$MRTS_{xy} = -\frac{dy}{dx}\bigg|_{Q=\bar{Q}} = \frac{\partial Q/\partial x}{\partial Q/\partial y}$$

3. 替代彈性公式

$$\sigma = \frac{d \ln(x/y)}{d \ln MRTS} = \frac{MRTS}{x/y} \cdot \frac{d(x/y)}{d(MRTS)}$$

4. 齊次函數判定

若 $f(tx) = t^k f(x)$ 對所有 $t > 0$ 成立，則 $f$ 為 $k$ 次齊次函數
